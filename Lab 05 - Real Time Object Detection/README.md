# Lab 05 – Real-Time Object Detection

This lab demonstrates real-time object detection using the **Arduino UNO Q** and **Arduino App Lab**.

The application processes live camera input and identifies objects by displaying their labels, confidence scores and bounding boxes.

## Concepts Explored

- Edge AI
- Computer vision
- Real-time image processing
- Object detection
- Local AI inference

## Result

The object-detection model successfully identified objects visible through the camera and displayed the detection results in real time.
